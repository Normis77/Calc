# Requisitos para ejecutar el programa

Para ejecutar correctamente el programa, es necesario instalar previamente Java y configurar un entorno básico de desarrollo.

## 1. Instalar Java (JDK)

Debe instalar el **JDK (Java Development Kit)**, que incluye las herramientas necesarias para compilar y ejecutar programas en Java.

Descargue e instale **JDK 21 para Windows** desde el sitio oficial de Oracle:

https://www.oracle.com/java/technologies/downloads/#jdk21-windows

> Se recomienda descargar la versión **Windows x64 Installer (.exe)** para facilitar la instalación.

Una vez descargado:

1. Abra el instalador.
2. Siga los pasos del asistente.
3. Complete la instalación con la configuración predeterminada.

---

## 2. Instalar Visual Studio Code

Desde **Microsoft Store**, descargue e instale **Visual Studio Code**.

También puede descargarlo desde su sitio oficial:

https://code.visualstudio.com/

---

## 3. Instalar extensiones de Java en Visual Studio Code

Después de instalar Visual Studio Code, abra el programa y diríjase a la sección de extensiones.

Instale las siguientes extensiones publicadas por Microsoft:

- **Extension Pack for Java**
- **Java**
- **Java Run**
- **Code Runner**

> Estas extensiones permiten compilar, ejecutar y trabajar correctamente con proyectos Java dentro de Visual Studio Code.

---

## 4. Verificar instalación

Una vez completada la instalación, reinicie el equipo (opcional, pero recomendado) y abra el proyecto en Visual Studio Code.

Con esto, el entorno quedará listo para ejecutar el programa.
